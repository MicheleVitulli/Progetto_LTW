Linguaggi e tecnologie per il web

L'obiettivo del corso è di introdurre alla programmazione lato client sul World Wide Web.
Nella prima parte, il corso introduce i linguaggi standard per la rappresentazione dell'informazione sul Web, in particolare HTML e XML, e il Document Object Model (DOM) di HTML e XML.
Nella seconda parte, il corso presenta JavaScript, il linguaggio principale per la programmazione lato client sul World Wide Web, e alcune librerie e framework per JavaScript.

Linguaggi di markup. HTML.

HTML è l’acronimo di HyperText Markup Language, è un linguaggio di markup (di ‘contrassegno’ o ‘di marcatura’), che permette di indicare come disporre gli elementi all’interno di una pagina.

Programmazione Web. Javascript.

In questo corso esploreremo gli elementi di base del linguaggio, dalla dichiarazione delle variabili ai tipi di dato, dalle istruzioni alle funzioni, analizzando la sua relazione “storica” con l’HTML.

Framework e librerie.Bootstrap e JQuery.

Bootstrap è una raccolta di strumenti liberi per la progettazione di interfacce. jQuery è una delle librerie più conosciute di JavaScript, studieremo il loro utilizzo per uno sviluppo web più flessibile e moderno.
